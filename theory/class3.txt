What is JSX?
    JSX is a code which looks syntactically like HTML but its not HTML.
    Behind the scenes, JSX will be converted to react code by using Babel and then it will be converted to HTML(DOM)

Superpowers of JSX
    Readablitiy, less code, easy to use

Role of type attribute in script tag? What options can I use there?
    Type attribute specifies the type of script and it identifies the content between <script> and </script>
        module, async, defer are some options

{TitleComponent}  vs {<TitleComponent/>} vs {<TitleComponent></TitleComponent>} in JSX

first one is a react element

second one is a functional component 

third one is also a functional component
