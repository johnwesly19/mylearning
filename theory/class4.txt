Why do we need keys in React? When do we need keys in React?
    We need keys to avoid re-rendering of whole thing. When a key is specified, only that node will be updated.
    we need keys when we are updating similar kind of elements like 10 div tags updating at a time.
    react will be confused which part to update, so it will re-render everything.
